[Thực hành] Luyện tập với cấu trúc switch-case
Bài 2: Viết lại cấu trúc if sau thành cấu trúc switch
Hồ Đăng San - C0419H1
Đề Bài let a = prompt('a?', '');

if (a === 0) {
 alert(0);
}

if (a === 1) {
 alert(1);
}

if (a === 2 || a === 3) {
 alert('2,3');
}
Bài Làm let a = parseInt(prompt('a?', ''));
switch (a) {
 case 0:
  alert(0);
  break;
 case 1:
  alert(1);
  break;
 case 2:
 case 3:
  alert('2,3');
  break;
}
